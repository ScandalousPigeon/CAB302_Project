# Project Requirements

## Project Name
[App Name]

## Purpose
This application is designed to help beginner programmers learn coding through guided support rather than solutions.

## Functional Requirements
1. The system shall provide programming problems for users to solve.
2. The user shall be able to view a problem description and requirements.
3. The user shall be able to write and submit code within the application.
4. The system shall evaluate submitted code using test cases.
5. The system shall display whether the submitted solution passes or fails.
6. The system shall provide hints related to the user’s current problem.
7. The system shall provide pseudocode or step-by-step guidance instead of full solutions.
8. The system shall recommend relevant learning resources based on the problem or error.

## Non-Functional Requirements
1. The application should be easy to use for first-time users.
2. Feedback should be clear and easy to understand.
